# iCIDER: Inducible Circuits for Interchangeable Design and Engineering Post-Ripening

This repository contains the design files, models, and tools for the iCIDER project.

## Repository Structure

- **[Parts List PDF](./parts-list/main.pdf)** - Parts list documentation
- **[Poster References PDF](./poster-references/references.pdf)** - Poster references
- **[cell-designer-files/](./cell-designer-files)** - CellDesigner XML files for metabolic pathways and models
- **[arabidopsis-codon-translator.r](./r-scripts/arabidopsis-codon-translator.r)** - R script for translating protein sequences into plant-optimized DNA
- **[figures/](./figures)** - Figures and images
